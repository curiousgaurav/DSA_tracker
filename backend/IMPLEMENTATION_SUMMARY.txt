⚙️ HOW TO USE
════════════════════════════════════════════════════════════════════════════

Method 1: CLI Tool (Recommended)
──────────────────────────────────────────────────────────────────────────
  node migrate-problems.js all        → Import all problems (15-30 mins)
  node migrate-problems.js easy       → Import easy problems (2-5 mins)
  node migrate-problems.js medium     → Import medium problems (5-10 mins)
  node migrate-problems.js hard       → Import hard problems (3-8 mins)
  node migrate-problems.js stats      → Show statistics
  node migrate-problems.js clear      → Clear all problems

Method 2: Automated Setup Scripts
──────────────────────────────────────────────────────────────────────────
  Windows:  QUICKSTART.bat
  Mac/Linux: bash QUICKSTART.sh

Method 3: HTTP API Endpoints (Admin Only)
──────────────────────────────────────────────────────────────────────────
  POST   /api/import/import-problems              → Import all
  POST   /api/import/import-problems/:difficulty  → Import specific
  GET    /api/import/import-stats                 → Get statistics
  DELETE /api/import/clear-problems               → Clear data

Method 4: Custom Code
──────────────────────────────────────────────────────────────────────────
  const { importAllProblems } = require('./services/problemImportService');
  await importAllProblems();
